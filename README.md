# img2ASCII
一个将图片二值化后转换为八点式盲文的程序 A program transfering image into Braille
## 使用方法
将你要转换的图片改名为test.png后置于.py文件同级目录下即可。您也可以根据需要调整`threshold`以使得ASCII显示的图像更清晰（一般来说，更小的`threshold`会使得转换后的图像上点更少，比如下降到10以下时，可能整个ASCII画画面都是空白的）。
## 示例
将本项目中的test.png投入程序，并将`threshold`参数调整为90后运行，会在产生Test1-4.png的中间图像后，在命令行上打印出如下内容：
